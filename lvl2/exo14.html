D'ailleurs parlons immédiatement de ces choses étranges ! Le nom de cette écriture est "Arrow function" ; mais nous allons commencer juste par des fonctions plus classiques ! Définissez une fonction nommée "helloWorld" qui affiche dans la console "Hello World from a function !" puis faites en sorte d'appeler cette fonction afin que ce message s'affiche dans la console !
Bonus, trouvez un moyen de faire cela avec ce qu'on appelle une IIFE
